2. COCO

  • coco/

    • annotations/ #该文件下共六种json文件，此处只取两种

      • instances_train2017.json # instances：为目标检测与实例分割的标注文件
      • instances_val2017.jsonjson文件主要包含以下几个字段

    json文件主要包含以下几个字段

    {
    “info”: info, # dict 数据集描述

    “licenses”: [license], # list ，内部是dict

    “images”: [image], # 主要包括file_name, height, width，id是图片的唯一标识

    “annotations”: [annotation], # list ，主要包括检测与分割的标注信息

    “categories”: # list ，内部是dict id, 主要字段包括image_id, category_id, segmentation, num_keypoints, area, iscrowd, keypoints，bbox等
    }

    • images/
      • train2017/
        • .jpg
      • val2017
        • .jpg
